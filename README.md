# Pine Script VS Code Extension – Feedback

This is the **public feedback and issue tracker** for the [Pine Script](https://www.tradingview.com/pine-script-docs/) VS Code extension.

## What is this repository for?

Use this repository to:

- 🐛 **Report bugs** – something broken or behaving unexpectedly
- 💡 **Request features** – ideas or improvements you'd like to see
- 📣 **Share general feedback** – anything else about the extension

> **Note:** This repository does not contain the extension source code. It is dedicated to community feedback only.

## How to submit feedback

1. Open the [Issues](../../issues) tab.
2. Click **New issue**.
3. Choose the appropriate template:
   - **Bug Report** – for problems and unexpected behaviour
   - **Feature Request** – for new ideas or enhancements
4. Fill in the form and submit.

Before opening a new issue, please search existing issues to avoid duplicates.

## Security

If you have found a **security vulnerability**, please do **not** open a public issue.  
Instead, follow the responsible-disclosure process described in [SECURITY.md](SECURITY.md).
